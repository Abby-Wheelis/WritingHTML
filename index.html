Basic d3 Methods

Select
The select() command takes a CSS selector as its input and hands off a reference to the first element in the DOM that matches that selector. Use select(“p”) to get the first paragraph element or selectAll(“p”) to get a reference to all of the paragraph elements in the DOM.
Append
The append() command creates a new DOM element that you specify and appends it just inside the end of the element it was handed by the select function. It then hands off a reference to this new element to the next method in the chain.
Text
The text() command inserts a string between the opening and closing tags of the current selection. If there was existing content, it will be overwritten.
Style
The style() command sets a CSS property on a given selection. It can also be attached to a function that creates an if then situation, which could prove useful in categorizing data.

Trying it Out
